# PRD-010 — Checkout via WhatsApp

| Campo | Valor |
|--------|--------|
| **Projeto** | ERP Comercial Grifres |
| **Versão** | 1.0.0 |
| **Status** | Aprovado |
| **Prioridade** | Crítica |

---

# 1. Objetivo

Implementar o processo de Checkout do ERP Comercial Grifres.

O checkout será responsável por coletar os dados do cliente, criar o pedido no ERP e, ao final, abrir uma conversa no WhatsApp da loja com uma mensagem pré-preenchida para confirmação do atendimento.

**O sistema não utilizará a API do WhatsApp.** A comunicação ocorrerá apenas por meio do link oficial `wa.me`.

O ERP será a fonte oficial de todas as informações do pedido.

---

# 2. Objetivos

- Criar fluxo completo de checkout.
- Coletar dados do cliente.
- Validar informações obrigatórias.
- Criar o pedido no banco de dados.
- Gerar número sequencial do pedido.
- Abrir automaticamente o WhatsApp da loja.
- Limpar o carrinho após a criação do pedido.

---

# 3. Escopo

Este PRD contempla:

- Página de Checkout.
- Cadastro dos dados do cliente.
- Cadastro do endereço.
- Seleção da forma de pagamento.
- Criação do pedido.
- Geração da mensagem para WhatsApp.
- Abertura automática do WhatsApp.

Não contempla:

- API do WhatsApp.
- Pagamento online.
- Cálculo automático de frete.
- Cupons.
- Cashback.

# 10. Criação do Pedido

Ao clicar em:

```text
Finalizar Pedido
```

O sistema deverá:

1. Validar o formulário.
2. Validar estoque.
3. Criar o pedido.
4. Criar os itens do pedido.
5. Gerar número sequencial.
6. Definir status inicial.
7. Salvar tudo no banco.
8. Gerar a mensagem.
9. Abrir o WhatsApp.
10. Limpar o carrinho.

---

# 11. Número do Pedido

Formato:

```text
#000001

#000002

#000003
```

O número deverá ser único.

---

# 12. Status Inicial

Todo pedido criado deverá iniciar com:

```text
Aguardando Confirmação
```

O pedido já deverá aparecer imediatamente no painel administrativo.

---

# 13. Mensagem do WhatsApp

Após salvar o pedido, abrir:

```text
https://wa.me/55XXXXXXXXXXX
```

Mensagem enviada:

```text
Olá!

Acabei de realizar um pedido pelo site da Grifres.

Pedido: #000154

Nome: João Silva

Telefone: (85) 99999-9999

Aguardo a confirmação.

Obrigado!
```

A mensagem não deverá conter a lista completa de produtos, pois todas as informações já estarão registradas no ERP.

# 22. Fora do Escopo

Não implementar:

- API do WhatsApp.
- Pagamento online.
- PIX automático.
- Integração com gateways.
- Frete automático.
- Cupons.
- Cashback.
- Programa de fidelidade.

---

# 23. Observações

O ERP Comercial Grifres será a única fonte oficial dos pedidos.

O WhatsApp funcionará exclusivamente como canal de comunicação entre o cliente e a loja, sem qualquer dependência de APIs externas.

Essa arquitetura permite uma operação simples, confiável e de fácil evolução futura, caso seja desejada uma integração oficial com a WhatsApp Cloud API ou outro canal de atendimento.
