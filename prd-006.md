# PRD-006 — Sistema de Produtos

| Campo | Valor |
|--------|--------|
| **Projeto** | ERP Comercial Grifres |
| **Versão** | 1.0.0 |
| **Status** | Aprovado |
| **Prioridade** | Crítica |

---

# 1. Objetivo

Implementar o módulo completo de Produtos do ERP Comercial Grifres.

Este módulo será o núcleo da loja virtual e do ERP, permitindo o cadastro, gerenciamento, organização e exibição de produtos tanto no painel administrativo quanto na loja virtual.

Todos os produtos cadastrados deverão ser imediatamente disponibilizados para exibição na loja, respeitando as configurações definidas pelo administrador.

---

# 2. Objetivos

- Desenvolver CRUD completo de produtos.
- Integrar produtos às categorias.
- Preparar estrutura para variantes.
- Preparar estrutura para controle de estoque.
- Permitir múltiplas imagens.
- Permitir produtos ativos e inativos.
- Disponibilizar produtos automaticamente na loja.

---

# 3. Escopo

Este PRD contempla:

- Cadastro de produtos.
- Listagem de produtos.
- Edição.
- Exclusão lógica.
- Visualização.
- Integração com categorias.
- Galeria de imagens.
- Organização dos produtos.

Não contempla:

- Variantes.
- Controle de estoque.
- Coleções.
- Promoções.
- Avaliações.
- Favoritos.

Esses recursos serão implementados em PRDs específicos.

# 6. Informações Comerciais

Cada produto deverá possuir:

```text

```

---

# 7. Galeria de Imagens

Cada produto poderá possuir múltiplas imagens.

Campos:

```text
id

produtoId

url

ordem

principal
```

Regras:

- mínimo de uma imagem;
- apenas uma imagem principal;
- ordenação manual.

Inicialmente utilizar URLs públicas.

---

# 8. Categorias

Todo produto deverá obrigatoriamente pertencer a uma categoria.

A categoria deverá ser selecionada através de um campo de busca.

---

# 9. Slug

O sistema deverá gerar automaticamente.

Exemplo:

```text
Camiseta Oversized Premium

↓

camiseta-oversized-premium
```

O administrador poderá editar manualmente.

O slug deverá ser único.

---

# 10. Produtos Ativos

Produtos inativos permanecerão cadastrados no ERP.

---

# 26. Observações

Este PRD representa o primeiro módulo comercial do ERP Comercial Grifres.

A partir da sua conclusão, a loja passará a operar com produtos reais cadastrados pelo administrador, eliminando completamente a dependência de dados fictícios e estabelecendo a base para os módulos de variantes, estoque, pedidos e gestão comercial.
